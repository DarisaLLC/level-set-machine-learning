stat_learn_level_set.init_funcs.random¶

class stat_learn_level_set.init_funcs.random.random(rs=None, reproducible=True)[source]¶

Initialize the zero level set to a sphere of random location and radius.

This initialization function works for all dimensions, so the intialization is a random circle in 2d, sphere in 3d, etc.

If the reproducible flag is set to True, then the same “random” initialization is produced when the same image is given as input.
