class stat_learn_level_set.neural_network.neural_network.neural_network(ninput, nhidden, rs=None, reg=0)[source]¶

Single hidden layer neural network with single output unit that computes the identity.

params: W, where w[i,j] = weight from input i to hidden unit j.
b, where b[j] = bias into hidden unit j. z, where z[j] = weight from hidden unit j to output unit. c, single scalar bias for output unit.

For a single vector input, the computation chain is: h = hidden_layer = tanh( dot(W.T, input) + b ) output = dot(h, z) + c
